# 操作系统

操作系统是管理硬件和软件的一种应用程序。操作系统是运行在计算机上最重要的一种**软件**。

**主要功能**

* 进程管理
  * 任务调度
  * 单核处理器下，为进程分配任务；
  * 多核处理器下，为进程分配任务外，处理器的调度、分配和回收等问题
* 内存管理
  * 管理内存的分配、回收，通过合理的页面置换算法进行页面的换入换出
* 设备管理：根据确定的设备分配原则对设备进行分配，使设备与主机能够并行工作
* 文件管理：组织和管理文件系统及其存储空间，为文件访问和文件保护提供更有效的方法及手段
* 提供用户接口：提供访问应用程序和硬件的接口

**主要目的**

* 管理计算机资源
* 提供界面
* 为其他软件提供服务

## 软件访问硬件

实质是一种 **I/O操作**，一般分为 **并行、串行**，对应 **并行接口**、**串行接口**。

**衡量原则**

* 数据传输速度快
* 系统开销及处理控制程序少
* **CPU** 等待时间少， **I/O** 设备尽可能忙

**控制方式**

* 直接访问（忙/等待方式）
  * 用户进程直接控制主存或 **CPU** 和 **I/O设备** 之间的信息传送
* 中断驱动
  * **I/O设备** 仅当操作 **正常结束** 或 **异常结束时** 向 **CPU** 发出中断请求，达到与 **CPU** 并行。
* **DMA** 直接存储器访问
  * **CPU** 发出传输指令，传输由 **DMA 控制器** 来实行和完成。
  * 方向、大小、地址都是 **CPU** 控制。
* 通道控制方式
  * 独立于 **CPU**，专门负责 **I/O** 的处理机
  * 有自己的通道指令，**CPU** 只负责启动，并在操作结束时向 **CPU** 发出中断信号。
